# Political Polarization in the United States
### Predicting Party Identification from Social Attitudes and Demographics Using ANES Data (1980s–2020s)

**Author:** Diana Havhun  
**Course:** MACS 30100 2 Machine Learning for Computational Social Science

## Overview
This project examines ideological sorting in the United States using machine learning. 
I build two parallel modeling tracks, one based on policy attitudes, one on demographic 
characteristics, and compare their predictive accuracy for party identification across 
five decades of ANES survey data.

## Files
- `ANES_Data_Polarization.qmd` — main analysis file (code + text)
- `references.bib` — bibliography

## Data
American National Election Studies (ANES) Cumulative Data File, 1948–2024.  
Available at: [electionstudies.org](https://electionstudies.org/data-center/)  
Due to file size, the data is not included in this repository and must be downloaded directly from the ANES website.
